Bye bye Wordpress, Hello Hugo

June 22, 2020

Joost

1-Minute Read

Twitter

Facebook

LinkedIn

Email

This site has been running on Wordpress ever since it conception back in 2011. As you might have noticed there is not a lot of new content added over the last couple of years. But that doesn’t mean that the site could run without maintenance. Especially since there are a lot of people that love to attack unpatched Wordpress installations.

With the preview release of Azure Static WebApps and the ever growing populairity of Static site generators like Hugo, I decided it was time for me to bite the bullet and migrate away from Wordpress.

As you can see the result is blazing fast. It’s just plain old HTML, CSS and JS running on an Azure Static Web App with a little help from Cloudflare.
